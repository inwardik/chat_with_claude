# Пример документа

Это пример документа для демонстрации работы системы индексации и поиска.

## Введение

Эта система позволяет индексировать ваши документы и затем задавать вопросы по их содержимому через Claude Code.

## Основные возможности

### 1. Семантический поиск

Система использует векторные эмбеддинги для понимания смысла текста, а не просто поиска ключевых слов.

### 2. Поддержка русского языка

Используется модель `intfloat/multilingual-e5-large`, которая хорошо работает с русским языком.

### 3. Локальное хранение

Все данные хранятся локально на вашем компьютере в ChromaDB.

## Технические детали

База данных ChromaDB хранит:
- Текстовые чанки из документов
- Векторные представления (эмбеддинги)
- Метаданные: источник, индекс чанка

## Примеры использования

Вы можете спрашивать у Claude:
- "Что написано про семантический поиск?"
- "Какие технологии используются?"
- "Как работает индексация?"

Система найдет релевантные фрагменты и предоставит их Claude для формирования ответа.
